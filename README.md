# Python-Based Expense Tracker

This expense tracker, which is based on Python, assists users in tracking and organizing their daily spending. Data is stored and retrieved using file handling.

## Attributes
Add to the cost
Examine every expense
Compute the total cost.
Keep data in a file.
A basic menu-driven user interface

## Technology Employed
"Python"
File Management

## How to Run
One. Click the project folder.
Two. Execute the Python file.
Expense.py in Python
Three. Adhere to the menu choices.

## 📌 Introduction
Users of this project can monitor their daily expenditures. It offers options to view and compute total expenses and saves expense information in a file. It helps with comprehending Python file handling.
